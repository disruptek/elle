## Summary
A static-site documentation generator written entirely in Elle, dogfooding 
the language to produce HTML documentation for the Elle programming language 
itself.

## What's included

**Rust changes:**
- Fix substring, char-at, string-length, and string-index to use character 
  indices instead of byte indices (fixes UTF-8 panics)

**Elle application (elle-doc/):**
- generate.l — 700+ line documentation generator
- JSON-based documentation source files for 5 pages
- Generates responsive HTML with dark/light mode (CSS prefers-color-scheme)
- Inline markdown formatting: **bold**, *italic*, `code`, [links](url)
- Content blocks: paragraphs, code blocks, tables, lists, blockquotes, notes
- Navigation sidebar with active page highlighting

**Documentation pages:**
- Home — introduction and feature overview
- Getting Started — installation and first steps
- Language Guide — data types, functions, control flow
- Standard Library — function reference with tables
- Examples — real-world code examples

**Pain points discovered (dogfooding value):**
- set! does not work inside lambda bodies (#106)
- map/filter/fold don't support closures (#99)
- try/catch doesn't catch runtime errors (#100)
- import-file doesn't properly export definitions to the importing scope
- let with 5+ bindings has variable resolution issues
- String primitives used byte indices instead of character indices (fixed)
